# Semantic Search Chatbot with LlamaIndex and OpenAI

This project implements a semantic search chatbot that indexes documents and allows users to query information using natural language. It uses LlamaIndex for document indexing and OpenAI's GPT-3.5-turbo for generating responses.

## Features
- **Document Indexing**: Indexes documents for semantic search using LlamaIndex.
- **Prompt Helper**: Configures input chunking for efficient processing and response generation.
- **Query Engine**: Allows users to query indexed documents and receive contextually relevant responses.
